About This App

This demo shows Q&A examples grouped by agent stage, but the production app is a batch pipeline — input a business context, four agents run in sequence, output is a 12-section executive market entry report in 10–15 minutes.

The real 4-agent pipeline:
1. Research Agent — scans market, identifies top 8 trends
2. Competitor Intel Agent — maps 10+ competitors & capabilities
3. Opportunity Agent — scores white-space on a 2×2 matrix
4. Strategy Agent — writes 12-section report + PDF

Dataset powering this demo: 3,000 Malaysian fruit retail transactions (September 2024, 21 SKUs, RM1.88M revenue).
