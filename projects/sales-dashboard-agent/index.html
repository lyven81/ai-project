üìñ About

                

                    The Sales Dashboard Agent is an intelligent analytics tool that leverages Google Gemini AI
                    to convert natural language questions into executable database queries. Simply upload your
                    sales data, ask questions in plain English, and receive instant insights with visualizations.
                

                

                    Built with modern AI technology, this agent understands context, generates safe Python code,
                    and provides business-friendly answers. Whether you're analyzing revenue trends, identifying
                    top customers, or evaluating product performance, the Sales Dashboard Agent makes data
                    analysis accessible to everyone.
