About This App

This demo shows Q&A examples grouped by agent stage, but the production app is a batch pipeline — input a Bursa ticker + its latest quarter report, five agents run in sequence, output is an analyst-ready report covering screening, fundamentals, moat, dividends, and the investment call.

The real 5-agent pipeline:
1. Screening Agent — basic health checks
2. Fundamentals Agent — margins & growth
3. Moat Agent — competitive position
4. Dividends Agent — payout sustainability
5. Reporting Agent — synthesis + investment call

Source report powering this demo: 99 Speed Mart Retail Holdings Berhad (Bursa: 99SMART) — Q3 2025 Unaudited Interim Financial Report (period ended 30 Sept 2025).
