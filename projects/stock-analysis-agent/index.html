What is Stock Analysis Agent?

A fully automated multi-agent investment analysis pipeline designed for retail investors analyzing Malaysian stocks for long-term dividend-focused investments. Instead of manually reviewing financial statements, specialized AI agents handle screening, analysis, and reporting.

Usage & Installation

Installation

# Install dependencies
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env
# Edit .env and add your API keys

Single Stock Analysis

from stock_analysis_agent import run_stock_analysis_pipeline

# Analyze Bursa Malaysia
results = run_stock_analysis_pipeline('1818.KL')

print(f"Report: {results['report_path']}")
print(f"Chart: {results['chart_path']}")

Stock Discovery

from stock_analysis_agent import discover_stocks

# Discover stocks matching criteria
results = discover_stocks(
    criteria={
        'max_price': 1.00,
        'min_pe': 4,
        'max_pe': 15,
        'min_roe': 5,
        'min_dividend_yield': 1
    },
    analyze_top_n=3
)
