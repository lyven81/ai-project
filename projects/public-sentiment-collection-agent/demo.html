About This App

This demo shows Q&A examples grouped by agent stage, but the production app is a batch pipeline — input a brand or keyword, five agents run in sequence, output is a packaged voice-of-customer report with sentiment breakdowns and action priorities.

The real 5-agent pipeline:
1. Listening Agent — scans social channels, collects mentions
2. Sentiment Agent — classifies positive / negative / neutral
3. Visualization Agent — builds sentiment charts
4. Export Agent — structures report-ready data
5. Packaging Agent — final report + recommendations

Dataset powering this demo: 500 coffee shop reviews (Jan–Mar 2024) across Twitter, Instagram, Facebook, and Google.
