🍜 Sarawak Laksa

A curated guide to Kuching's most iconic dish

HTML5 CSS3 Vanilla JS IntersectionObserver Static Site
🚀 Launch Live Demo 📁 View Source Code

📋 Project Overview & Problem Statement

Challenge: Sarawak Laksa is one of Malaysia's most celebrated dishes, yet no single well-designed digital reference brings together its history, recipe, cultural context, and where to eat it — in one readable, navigable page.

Solution: A rich, scroll-driven reference site built entirely in HTML, CSS, and vanilla JavaScript. It covers everything a food lover, tourist, or diaspora Sarawakian would want to know — from the paste ingredients to Anthony Bourdain's famous quote — in a clean, readable format with smooth animations.

Key Benefits

  • Comprehensive: 7 sections covering origin, ingredients, recipe, variations, stalls, culture, and people
  • No login required: Fully public, no account or API key needed
  • Smooth experience: Fade-in animations, active nav tracking, sticky top navigation
  • Useful for tourists: Includes real stall names, opening hours, halal/non-halal status
  • Useful for home cooks: Full 4-phase recipe with pro tips and paste brand recommendations

🖥️ Application Features

🌿 Origin & Overview

Explains what Sarawak Laksa is, how it differs from Penang Asam Laksa and Singapore Laksa, with a quick-facts card and comparison table.

🫚 Ingredient Breakdown

Groups ingredients into Paste, Broth, and Toppings. Each card includes the ingredient, quantity, and its role in the dish.

📖 Step-by-Step Recipe

4-phase recipe (Paste → Broth → Toppings → Assemble) with numbered steps, pro tips, and store-bought paste alternatives.

🗺️ Regional Variations

Covers Kuching, Sibu, Miri, Home-Style, and Restaurant styles — each with flavour profile tags.

📍 Where to Eat Guide

6 real Kuching stalls with address, opening hours, halal status, and what makes each one worth visiting.

🎭 Culture & People

6 cultural context pieces (breakfast ritual, kopitiam setting, diaspora longing) and profiles of key figures including Anthony Bourdain and Gordon Ramsay.

⚙️ Technical Architecture & Implementation

Frontend Stack

HTML5 CSS3 Custom Properties Vanilla JavaScript Google Fonts (Playfair Display + Inter) IntersectionObserver API CSS Grid & Flexbox

Architecture Highlights

  • Fully static — no server, no database, no dependencies
  • Sticky top navigation with active section highlighting via IntersectionObserver
  • Fade-in card animations triggered on scroll using IntersectionObserver with threshold and rootMargin
  • Responsive grid layouts — 4-col ingredient cards collapse to 2-col on mobile
  • All content is HTML — no external data files or API calls at runtime
  • Knowledge files (Markdown) in /knowledge/ served as source documentation

🚀 Deployment

Hosted as a static site on GitHub Pages. No build step required — open index.html directly in any browser.
