Wiseman · 智者

Daily cosmic reading in Mandarin Chinese — Bazi, horoscope, planetary astrology, and Chinese zodiac

Node.js Express Claude API ES Modules

About This Code Showcase

This curated selection highlights how Wiseman generates personalized daily cosmic readings by combining five astrological systems — Bazi, Western horoscope, planetary positions, Chinese zodiac, and Mayan astrology — into a single AI-interpreted narrative.

The app uses a modular API architecture where each astrological system has its own module under apis/, and readings are cached per day to avoid redundant LLM calls.

Core: Express Server & API Routes

The server exposes three main endpoints: /api/reading for daily cosmic readings, /api/chat for follow-up conversation with Wiseman, and /api/profile for the one-time personality profile. Each reading is cached by date so the same day always returns consistent guidance.

server.mjs — Reading & Chat Routes
import { generateReading, buildChatSystemPrompt } from './apis/reading.mjs' import { generateProfile } from './apis/profile.mjs' import { callClaudeChat } from './lib/llm.mjs' import { config } from './wiseman.config.mjs' // GET /api/reading — returns today's reading (cached or fresh) app.get('/api/reading', async (req, res) => { const dateStr = req.query.date || getTodayStr() const reading = await generateReading(dateStr) res.json({ ok: true, reading }) }) // POST /api/chat — chat with Wiseman about today's reading app.post('/api/chat', async (req, res) => { const { message, history, readingDate } = req.body const dateStr = readingDate || getTodayStr() const reading = await generateReading(dateStr) const systemPrompt = buildChatSystemPrompt(reading) // Build message history for Claude const messages = [] if (history && Array.isArray(history)) { for (const h of history) { if (h.role && h.content) messages.push({ role: h.role, content: h.content }) } } messages.push({ role: 'user', content: message }) const reply = await callClaudeChat({ systemPrompt, messages, maxTokens: 512 }) res.json({ ok: true, reply }) })

Technical Implementation Notes

Key Design Decisions

  • Five astrological systems: Bazi (八字), Western horoscope, planetary positions, Chinese zodiac, and Mayan calendar are each handled by a dedicated module under apis/
  • Daily caching: Readings are saved as JSON files in runs/ by date, so the same day always returns consistent guidance without extra API calls
  • Persistent profile: The "Who Am I" personality profile is generated once and cached in profile.json — a deep character reading that never changes
  • Context-aware chat: Follow-up conversations receive today's full reading as system context, so Wiseman can reference specific cosmic influences in replies
  • All output in Mandarin: The entire reading and chat experience is delivered in Chinese, matching the traditional character of the astrological systems
